# Wayfinder Door Unlocker
A **[Wayfinder](https://github.com/Echoviax/wayfinder)** mod that Forces doors in *Neverway* to be open.  
Currently only works with NPC doors
## Installation
1. Download the latest releases from the releases tab
2. Drag to your `Mods` folder
